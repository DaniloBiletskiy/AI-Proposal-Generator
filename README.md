# AI Proposal Generator

AI-powered web application that analyzes freelance job descriptions and generates professional proposals for freelancers.

## Features

- AI job description analysis
- Project complexity estimation
- Budget estimation
- Timeline prediction
- Professional proposal generation
- Client question generation
- Custom skills input
- Analysis history
- Responsive dark UI

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- Groq API
- Render

## Live Demo

https://ai-proposal-generator-zyc0.onrender.com
## How It Works

1. Paste a freelance job description.
2. Add your skills.
3. Generate analysis.
4. Receive:
   - Project summary
   - Complexity score
   - Budget estimate
   - Timeline estimate
   - Professional proposal
   - Questions for the client
